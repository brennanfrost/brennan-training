# Brennan Frost — Training Plan App

Personal training dashboard for the **Salt Lake Criterium, August 15**.

## What's inside

- **Training Calendar** — 17-week plan from April to August with all races marked. Click any workout block to mark it complete. Completions save automatically in your browser.
- **Gym Plan** — Phased lifting plan (Base → Power → Taper) with progressive loads for squat, bench, and deadlift. Click rows to mark sessions done.
- **Meal Plan** — Daily schedule timed around 6:30am gym and 4pm ride. Includes dinner rotation and weekly grocery list.
- **Race Day** — Nutrition timing, tactics, and pre-race week strategy for August 15.

## How to use

### Option 1 — Open directly
Just open `index.html` in any browser. No server needed, no installs.

### Option 2 — GitHub Pages (recommended)
1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)`
4. Your app will be live at `https://yourusername.github.io/repo-name`

## Features
- Dark mode UI
- Click any workout to mark complete (saves to browser localStorage)
- Live countdown to race day
- Phase filtering on calendar and gym plan
- Fully mobile responsive
